# rock-paper-scissors
Its a fun Rock Paper Scissors game designed using basic html, css and javascript
THE FILE STRUCTURE IS AS FOLLOWS:
  1)index.html  --> the basic html 
  2)css.html   --> stylesheet for the above
  3)images/  --> contains the required images in png format

About:
  it is a single player version where you face off against a "computer" which is controlled by a random number generator.
  it contains score tracking for both the player and computer.
  clickable image buttons for Rock, Paper, and Scissors.
  a reset button to reset score.

How to run:
  1) download or clone this repo.
  2) open index.html in your browser.
  3) enjoy the game!
